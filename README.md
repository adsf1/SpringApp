## Endpoints

| HTTP Request Method | Endpoint      |
|---------------------|---------------|
| GET                 | /courses      |
| POST                | /courses      |
| GET                 | /courses/{id} |
| PUT                 | /courses/{id} |
| DELETE              | /courses/{id} |
