## Endpoints

| HTTP Request Method | Endpoint      | Goal                   |
|---------------------|---------------|------------------------|
| GET                 | /courses      | Get all courses        |
| POST                | /courses      | Create course          |
| GET                 | /courses/{id} | Get specific course    |
| PUT                 | /courses/{id} | Update specific course |
| DELETE              | /courses/{id} | Delete specific course |

<hr>
